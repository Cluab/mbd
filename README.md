# MBD - Muslim Business Directory

A Flutter application for iOS and Android platforms that helps newcomer Muslims in Canada locate and connect with Muslim-friendly businesses and services.

## Project Overview

MBD serves as a specialized directory (similar to Google Maps) that focuses on the Muslim community in Canada. The app helps users easily identify and locate:

- Halal restaurants and food services
- Mosques and prayer spaces
- Islamic centers and educational institutions
- Muslim-owned businesses
- Professional services catering to Muslims

The primary goal is to provide newcomer Muslims with a valuable resource that simplifies finding essential services and helps them feel more connected to their community as they settle in Canada.

## Project Setup

### Prerequisites
- Flutter SDK (version 3.29.3 or newer)
- Android Studio (for Android development)
- Xcode (for iOS development, Mac only)
- Git

### Getting Started

1. Clone the repository:
```bash
git clone [repository-url]
cd mbd
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
# For Android
flutter run -d android

# For iOS (Mac only)
flutter run -d ios
```
